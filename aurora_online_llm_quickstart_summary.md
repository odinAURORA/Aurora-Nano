# Aurora Online LLM Quickstart Pack

## Status
Prepared a small package for people using ChatGPT, Claude, Gemini, Grok, Perplexity, or other online LLMs.

## Contents
- claim extraction prompt
- short prompt
- JSON schema
- example ticket
- example model output
- online LLM usage guide
- Hugging Face README addon

## Purpose
Make Aurora easy to use without local model setup.

## Main Message
Your model does the language part. Aurora does the consistency-checking part.
